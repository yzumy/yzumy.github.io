このディレクトリには以下の音声ファイルを配置してください：

1. click.mp3 - 通常のクリック音
2. click_accent.mp3 - アクセント付きのクリック音
3. woodblock.mp3 - ウッドブロック音
4. woodblock_accent.mp3 - アクセント付きのウッドブロック音
5. cowbell.mp3 - カウベル音
6. cowbell_accent.mp3 - アクセント付きのカウベル音
7. hihat.mp3 - ハイハット音
8. hihat_accent.mp3 - アクセント付きのハイハット音

音声ファイルが見つからない場合、アプリケーションは Web Audio API を使用して代替音を生成します。
